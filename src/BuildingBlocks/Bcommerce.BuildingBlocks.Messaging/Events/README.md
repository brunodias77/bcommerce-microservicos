# Shared Event Contracts

Esta pasta destina-se a armazenar os contratos de eventos compartilhados (DTOs) que são utilizados por múltiplos microsserviços. 

Recomenda-se publicar esses contratos como um pacote NuGet separado ou mantê-los aqui referenciados pelos projetos.

Exemplos de eventos que ficariam aqui:
* `OrderCreatedEvent`
* `PaymentProcessedEvent`
* `StockReservedEvent`
